# How to install: 
## Backend
Requirements: Python 3.7, Pipenv (https://pipenv-fork.readthedocs.io/en/latest/install.html)

```
git clone https://github.com/tap2open2/LaTraviata.git
cd LaTraviata && pipenv install && pipenv shell
python manage.py
```
